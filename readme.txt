D Slider Plugin for WordPress allows creation and management of simple image slideshows.

== Description ==
This Plugin is developed for them who are looking for a simple but gorgeous slider plugin like a carousel slider hopefully it will solve your issue.

Don\'t wait.. install it and check how it works and how it looks.

== Installation ==
  **Upload d-slider.zip to the /wp-content/plugins/ directory.

  **Activate the plugin through the \'Plugins\' menu in WordPress.

  **You will see a new menu in the dashboard \"Slider\"... add a Slider and name it and set images for it.

  **After Create Slider go back to Slider and find shortcode at the right sight.

  **now just add this shortcode anywhere in the page, post or widgets.

        [d_slider id=\'2654\' /]
